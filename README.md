# lfetch
A fetch App written in lua

## Install
```bash
git clone https://github.com/ysfgrgO7/lfetch.git
cd lfetch
sudo make install
```

## Ascii support
  - Ubuntu - Arch - Manjaro - Gentoo - Fedora - Debian - Alpine - Void - Android
  - Any other distro will print a generic penguin
